# Good Sole 
Good Sole is an application that sells shoes ranging from track & field to formal wear.

## Detailed Description

This is an application that caters to athletes as well as sneaker heads. Good Sole is a shoe retail application which sells a variety of footwear,
whether it's for sporting or casual wear we have it! Good Sole will be user friendly giving every user an easy to follow GUI with categories which
will make for easier shopping!

### Project Introduction 

- Good Sole is a online shoe store which has a wide variety of footwear available to users for delivery.
- The application design will be simple and easy to use allowing for easy maneuverability.
- There will be categories for the user to choose from.
- Good Sole will eleminate the process of going to the store with shoe delivery being its main objective.
- Good Sole is in it's development phase.
